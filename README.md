# DT money
Simple project to administrate the personal finances, using React, Node and Styled Components library.

# First running
`yarn start`

## Directory structure
- public
    - index.html
- src
    - assets
    - components
        - Dashboard
        - Header
        - Summary
        - TransactionsTable
    - styles
        - global.ts
    - index.tsx
    - App.tsx

## Dependencies
* styled-components
* @types/styled-components
