# DT money
Simple project to administrate the personal finances, using React, Node and Styled Components library. The MirageJS is used to mock the API.

# First running
```
yarn start
```

## Directory structure
- public
    - index.html
- src
    - assets
    - components
        - Dashboard
        - Header
        - Summary
        - TransactionsTable
    - services
        api.ts
    - styles
        - global.ts
    - index.tsx
    - App.tsx

## Dependencies
```
yarn add styled-components
yarn add --saveDev @types/styled-components
yarn add miragejs
yarn add --saveDev @types/miragejs
yarn add axios
```
